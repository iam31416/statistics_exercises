# Statistics Exercises – R

This repository contains exercise sheets (PDF statements) and R skeleton scripts for the Statistics course.

## Units
1. Descriptive Statistics
2. Statistical Models
3. Time Series
4. Probability
5. Random Variables
6. Distributions
7. Statistical Inference

## Structure
Each unit contains:
- statements/  (PDF exercise sheets)
- R/           (R skeleton scripts)

Solutions are not provided in this repository.
